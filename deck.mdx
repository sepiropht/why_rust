## Comment rust réussi cet exploit ?



---
Dans la plupart des languages
``` 
  main()
  function main() {
    let foo = [1, 2]
    let bar = foo
    console.log(bar) // [1, 2]
    console.log(foo) // [1, 2]
  }
  ```




---



  En rust ...
  ``` 
  fn main () {
    let foo = [1, 2]
    let bar = foo
    println!("{}", bar) // [1, 2]
    println!("{}", foo) // Erreur de compilation !!! La variable dans foo a été move 
  }
  ```
